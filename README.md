# vfs_shell

A mock CLI shell for reading and acting upon a virtual file system.

## Usage

Display the help message:

```cli
shell --help
```

Output the VFS:

```cli
shell -f file.vfs
```

List the files in the VFS:

```cli
shell -f vile.vfs --dir
```
